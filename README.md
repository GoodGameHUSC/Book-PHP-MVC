# PHP Book Store

- PHP MVC 
- Webpack, SASS
- Built-in server



#### Installation
  `composer install`
  `npm install`

#### Run local
  `composer dev` 
  or 
  `php -S localhost:8000 -t public`
  then open [localhost:8000](http://localhost:8000) to see website.

  Otherwise, you also run with web server.

#### Run Webpack (optional)
  `npm run watch`
